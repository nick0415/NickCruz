Conociendo WIX

WIX, es una plataforma online que ofrece una serie de herramientas para la creación de sitios web profesionales de forma rápida y sencilla, ya que cuenta con funciones preestablecidas y la elección de plantillas personalizables, se puede construir una web en muy poco tiempo y con unos resultados de gran calidad.
